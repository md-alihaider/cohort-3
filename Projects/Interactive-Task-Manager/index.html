Rendering Explanation

  • Browser parses HTML and creates the DOM Tree.
  • Browser parses CSS and creates the CSSOM Tree.
  • DOM + CSSOM combine to form the Render Tree (only visible elements).
  • Layout calculates exact positions and sizes on the viewport.
  • Paint draws the actual pixels (colors, text, shadows) on the screen.
  • Composite merges overlapping layers together.
